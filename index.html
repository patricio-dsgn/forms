Un formulario web dentro de una página web permite al usuario introducir datos los cuales son enviados a un servidor para ser procesados.

Los formularios web se parecen a los formularios de papel porque los internautas llenan dichos formularios usando casillas de selección, botones de opción, o campos de texto.

+ una técnica de desarrollo

AJAX

HTML5

Lenguaje de Marcado de Hipertexto

Es el código que estructurar y desplega una página web y sus contenidos.

Párrafos, listas, imágenes y tablas, etc.

CSS3

Hojas de estilo en cascada

Es un lenguaje para la composición y estructuración de estilos en páginas web

Tipografias, colores, formas, etc.

JS

JavaScript

Es un lenguaje de programación que permite añadir características interactivas a sitios web.

Juegos, eventos al presionar botones, o ingresar datos en formularios, animaciones, etc.

PHP

PHP

Es un lenguaje de programación que permite el desarrollo web o aplicaciones web dinámicas, y se puede incrustar en lenguaje HTML.

Recopilar datos de formularios, generar páginas con contenidos dinámicos, enviar, recibir y procesar datos.

AJAX

JavaScript asíncrono y XML

Es un conjunto de técnicas de desarrollo web que permiten que las aplicaciones web funcionen de forma asíncrona, es decir, procesando cualquier solicitud al servidor en segundo plano.

Realiza acciones sin recargar la página: Enviar emails, crear y modificar archivos, etc.
